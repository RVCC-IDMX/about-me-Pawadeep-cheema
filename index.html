Animated Button with Neon Glow and 3D Press Effect

As a student of Mathematics and Data Science, I have recently started exploring the world of web development. In my latest mini project about me, I created a neon-animated button with a hover glow effect and a 3D press animation. The main idea behind this project was to understand how styling and animation works.

To build this animated button, I used HTML structure and CSS for styling and animation. The neon glow effect was acheived using box-shadow properties, and transitions made the hover effect smooth. For the 3D press effect, I adjusted the trasform and added a slight downward movement when the button is clicked, giving it realistic feeling. This animation helped me understand CSS transitions, hover states, and user interaction design.

Overall, creating this neon button was a fun and valuable learning experience. It improved both my creativity and technical skills. I look forward to building more interactive and user-friendly designs.
